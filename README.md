## Como executar?

- Clone esse repositório:

```bash
$ git clone https://github.com/cp-yago/liven-market.git
```

- Acesse o diretório do projeto

```bash
$ cd liven-market
```

- Instale as dependências:

```bash
$ yarn 
```

- Execute o projeto:

```bash
$ yarn start
```

- Para executar os testes:

```bash
$ yarn test
```
